# LIS545
LIS545 final project Github repository

This repository includes all materials submitted for the LIS545 final project. 
1. Final Report.txt includes a data and metadata profile regarding Hugo Darwood's "Epicurious - Recipes with Rating and Nutrition" dataset published on Kaggle; a repository profile which discusses the University of California Irvine's Machine Learning Repository as a good fit for the recipes dataset; a recommended data citation; and considerations for long-term preservation, human subjects, and copyright licensing. 
2. Metadata.xml includes a metadata file for the dataset generated by Microsoft Copilot and structured according to DataCite's metadata schema.
3. full_format_recipes.json.zip is a zipped file containing recipe data for over 20,000 recipes originally posted to Epicurious.
4. epi_r.csv.zip is a zipped file which includes only the titles, rating, and nutrition information from each recipe.
5. recipe.py is the original script used to create the dataset
6. utils.py is a script that was used to format the original data
